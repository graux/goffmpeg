FFMPEG wrapper written in GO

## Features

- [x] Transcoding
- [x] Streaming
- [x] Progress
- [x] Filters
- [x] Thumbnails
- [x] Watermark
- [ ] Concatenation
- [ ] Subtitles

## Dependencies
- [FFmpeg](https://www.ffmpeg.org/)
- [FFProbe](https://www.ffmpeg.org/ffprobe.html)

## Supported platforms

 - Linux
 - OS X
 - Windows

## Installation
Install the package with the following command:
```shell
go get github.com/graux/goffmpeg
```
